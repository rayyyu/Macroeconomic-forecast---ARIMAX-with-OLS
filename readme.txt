'Input_data.xlsx' is the input file for 'arimax_Finding_best_model.py', containing history value for each macroeconomic value of interest. And it generates 'ARIMAX_sh_best_models.xlsx', which contains the best ARIMAX models for each macroeconomic variable.


'ARIMAX_sh_best_models.xlsx', along with 'xtest_arimax' is the input for 'arimax_Finding_predicted_value', which generates predicted value for each macroeconomic variable.
